# 🔥 MemeForge

**MemeForge** is a modern, single-file Meme Generator web app built using HTML, CSS, and JavaScript.
Create high-quality memes instantly by adding your own text to random meme templates.

---

## 🚀 Features

* 🎨 Clean and modern UI with dark theme
* 🖼️ Random meme image generator
* ✍️ Add **Top Text** and **Bottom Text**
* 💡 Smart **context-based text suggestions**
* ⚡ Live preview (updates instantly while typing)
* 🔀 Generate new meme with same text
* ⬇️ Download meme as image
* 📱 Fully responsive design
* 🔄 Works even without API (fallback images included)

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Custom styling + variables)
* Vanilla JavaScript
* Canvas API (for rendering memes)
* Imgflip API

---

## 🔗 API Integration

* Imgflip Meme API
  https://api.imgflip.com/get_memes

### Includes:

* CORS proxy fallback
* Local fallback meme list (ensures app always works)

## 🎮 How to Use

1. Enter your **Top Text**  
2. Enter your **Bottom Text**  
3. (Optional) Add a **Context** (e.g. "Monday meetings")  
4. Click **⚡ Generate Meme**  

### Actions:
- 🔀 **New Image** → Change meme template  
- ⬇️ **Download** → Save your meme  

---

## 🧠 Smart Context Feature

MemeForge includes a simple rule-based system that:
- Detects keywords (e.g. *meeting, bug, coffee, exam*)  
- Suggests funny meme captions  
- Auto-fills text if fields are empty
